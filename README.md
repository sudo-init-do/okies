# 🎬 Okies Backend

This is the backend API for **Okies**, a short-form video and gifting social platform. Built with **NestJS** and **Firebase**, this backend handles user authentication, post interactions, gifting, and admin-level moderation tools.

---

## 🚀 Features

- 🔐 Firebase JWT Authentication
- 🎥 Video Post Uploading
- ❤️ Post Interactions (Like, Unlike, Comment)
- 🌹 Gifting System with Leaderboard
- 💸 Earnings & Withdrawals
- 🧑‍💼 Admin Controls (Approve/Reject Withdrawals)
- 🧊 Cooldown logic for gift spam prevention

---

## 🧠 Tech Stack

- **NestJS** (TypeScript)
- **Firebase Firestore**
- **Firebase Auth**
- **Firebase Admin SDK**
- **JWT + Custom Guards**
